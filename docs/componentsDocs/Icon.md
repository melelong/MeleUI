# Icon 图标

提供了一套常用的图标集合。

### 使用方法

直接通过设置类名为 `ml-icon-iconName` 来使用即可。例如：

<Icon-Demo1 />

---

::: details 查看代码

```vue
<template>
  <div class="demoBlock">
    <i class="ml-icon-edit"></i>
    <i class="ml-icon-share"></i>
    <i class="ml-icon-delete"></i>
    <ml-button type="primary" icon="ml-icon-search">搜索</ml-button>
  </div>
</template>
```

:::
