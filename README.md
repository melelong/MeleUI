# ml-ui(组件库)

### 快速开始

#### 1.安装组件库

```bash
npm i MeleUI
```

#### 2.引入组件库

```javascript
//  全部引入
import 'MeleUI/dist/theme-chalk/index.css';
import Ml from 'MeleUI';
Vue.use(Ml);

//  按需引用
import 'MeleUI/dist/theme-chalk/card.css';
import { MlCard } from 'MeleUI';
Vue.use(MlCard);
```
